https://dt207g-lab4-webbpla.onrender.com

TheGuestbook är en webbplats där användaren kan registrera ett konto och sedan använda det för att logga in. En inloggad användare kan posta inlägg i gästboken och även läsa. 
Om en användare inte är inloggad kan den endast läsa inlägg, men inte posta något. JWT'n sparas i webbläsarens localStorage och användas för att autentisera användaren vid anrop 
till skyddade resurserna i webbtjänsten.
